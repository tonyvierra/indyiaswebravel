# Explore With Indy — GitHub Pages Demo

This is a simple static multi-page website prototype for a travel brand.

## Files

- `index.html` — home page
- `about.html` — about page
- `services.html` — travel services page
- `destinations.html` — destination inspiration page
- `contact.html` — inquiry form prototype
- `styles.css` — shared styling

## How to publish with GitHub Pages

1. Create a GitHub repository.
2. Upload all files in this folder into the root of the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Save.
6. Open the GitHub Pages URL GitHub gives you.

## Important

GitHub Pages is static hosting. The contact form in this demo does not actually send messages.
For a real form, connect it to Formspree, Google Forms, Typeform, Wix Forms, or another form backend.
